### 🌟 Overview
A sleek, high-performance **Windows Forms** application built with C# that transforms plain text into 8-bit binary sequences in real-time. Designed with a custom **dark-themed UI** for a modern developer experience.

### ✨ Key Features
* **⚡ Real-time Conversion:** Instant text-to-binary transformation.
* **🌘 Dark Aesthetic:** Custom 4K dark-themed UI, comfortable for long study sessions.
* **🎮 Interactive Controls:** Functional 'Clear', 'Enable', and 'Disable' buttons for full UI control.
* **🛠 Built for Learning:** Part of my journey into Computer Science fundamentals and Data Structures.

### 🚀 How to Run
1.  **Clone** the repository:
    ```bash
    git clone [https://github.com/s-a-m-y1/ConvertToBinary-for-window-form.git](https://github.com/s-a-m-y1/ConvertToBinary-for-window-form.git)
    ```
2.  Open the `.sln` file in **Visual Studio**.
3.  Press **F5** to run the application.
